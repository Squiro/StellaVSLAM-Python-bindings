Pasos para compilar los bindings:

Si surge un error relacionado a eigen3, entonces se debe crear un link simbolico a eigen3:

sudo ln -s /usr/local/include/eigen3/Eigen /usr/include/Eigen

--

Modificacion del comando para que se pueda ejecutar con stella_vslam:

/usr/bin/g++ -O3 -Wall -shared -std=c++11 -fPIC $(python3 -m pybind11 --includes) -I/usr/local/include/stella_vslam/3rd/json/include/ -I/usr/local/include/stella_vslam/3rd/spdlog/include -I/usr/local/include/eigen3 -I/usr/local/include/opencv4 -I/usr/local/include/opencv2 -I/usr/local/include/pangolin ./stellavslam_bindings.cpp -o stella_vslam$(python3-config --extension-suffix) -lstella_vslam 

--

Si tira error de numpy, es porque hay que instalarlo:

pip install numpy
sudo apt-get install python-numpy
